# Chip Puzzle Game

1000개의 도전적인 스테이지가 있는 매칭 퍼즐 게임

## 기술 스택

- **React 18** + **TypeScript**
- **Vite** (빌드 도구)
- **Canvas API** (게임 렌더링)
- **GitHub Pages** (배포)

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 배포

```bash
npm run deploy
```

## 프로젝트 구조

```
src/
├── components/     # React 컴포넌트
├── hooks/          # Custom Hooks
├── services/       # 비즈니스 로직
├── utils/          # 유틸리티 함수
├── types/          # TypeScript 타입
├── constants/      # 상수
├── locales/        # 다국어 파일
└── styles/         # CSS 파일
```

## 기능

- ✅ Canvas 기반 게임 렌더링
- ✅ 1000개 스테이지 시스템
- ✅ 다국어 지원 (한국어, 영어, 중국어, 일본어)
- ✅ 반응형 디자인
- ✅ PWA 지원
